EVENTS

IRIC is collaborating with Team EXE and organising the following events this Nimbus.

Exeplore 4.0

This event requires a team of 2 members. Event will be conducted in 2 stages. First stage of event will consist of a preliminary round consisting of 15 questions based on a movie clip of duration 5-10 minutes. After that each participating team will be given a time of 15 minutes to solve the questions. The questions will be either MCQs or one word answer type. Second stage of event will be a puzzle solving event. Participants have to explore the way out by solving each puzzle correctly that will take them to another level of puzzle on a computer based interface. The one who solves the puzzles fastest or who scores highest will be declared as the winner.

Debugging 3.0

We will give some code to the user which will have some compilation and logical errors, like we will use some other file like an audio and image file and use it as the base to create the bugs we only have to get the encrypted error message, as done in the Udbhava.

Mackey Mackey 2.0

Makey-Makey is a simple electronic board that can send keyboard and mouse signals to a computer via everyday objects. Thus, we can turn anything into a touchpad and make computer Operation interactive. The Makey-Makey event will be held onsite during the three days of Nimbus itself featuring simple games that will be played using interactive consoles connected by the Makey- Makey console instead of keyboard or mouse.
